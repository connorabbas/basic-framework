# PHP Mini Framework

Key Features:
- Routing for GET and POST requests
- MVC architecture
- Class auto loading
- PDO database class
- Bootstrap, jQuery, and React included

starter content for env.php file (create in /app directory):
```
<?php
// Private ENV variables
define('DB_HOST', '127.0.0.1');
define('DB_USERNAME', 'root');
define('DB_PASSWORD', '');
define('DB_NAME', '');
?>
```
